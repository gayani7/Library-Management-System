# Library-Management-System
A simple Library Management System developed using Java (Swing GUI) and MySQL database.

## Technologies Used
- Java (JFrame,Swing)
- NetBeans IDE
- MySQL Database
- MySQL Connector/J (JDBC Driver)

## Features
- Login page
- Home page
- New Student Page
- New Book Page
- Print student records
- Issue Book Page
- Return Book Page
- Statistics Page
